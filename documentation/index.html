Quick Installation Guide

  1. Create Database & Import included SQL file that comes with the Edmy zip file.
  2. Open the terminal window into the edmy-react-next folder.
  3. Run npm install.
  4. Run npm run dev.
  5. Open the browser copy and paste the link- http://localhost:3000 in the browser address bar.
  6. The website will load, and the process is done!

Setup & Enable Development Environment

Application Install Preparation

  • Install Node.js: NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node.js.
  • Install WAMP for MySQL/PHPMyAdmin.
  • Create a Cloudinary Account And Follow the below screenshots.

    • - https://api.cloudinary.com/v1_1/cloud-name/image/upload

    - https://api.cloudinary.com/v1_1/cloud-name/video/upload

    Update the Cloudinary info next.config.js file

  • Create a Stripe Account.

    • Now from the application root directory, open the 'next.config.js' file using your code editor and update like below- 

    Copy                            CLOUDINARY_URL: "https://api.cloudinary.com......",
                            CLOUDINARY_VIDEO_URL: "https://api.cloudinary.com......",
                            STRIPE_SECRET_KEY: "sk_test_3D..........",
                            STRIPE_PUBLISHABLE_KEY: "pk_test_Zy........"

Finish Setup and Run the App

Use a command line tool i.e. PowerShell or terminal and navigate to the project(edmy-react-next) root folder. After that, you need to run the following commands-

npm install
npm run dev

It'll return something like this in the console:

>edmy-react-next@1.3.0 dev drive:\project

> next dev

ready - started on 0.0.0.0.3000, url: http://localhost:3000

[ event ] compiled successfully

Now, in the browser go to localhost:3000
